语法部分
box-shadow: none | [inset x-offset y-offset blur-radius spread-radius color] [,inset x-offset y-offset blur-radius spread-radius color]
  • inset: 阴影类型，可选值。如果不设置，默认投影方式是外阴影；如果取唯一值"inset"，就是设置内阴影
  • x-offset: 水平偏移量，正值在右边，负值在左边
  • y-offset:垂直偏移量，正值在下边，负值在上边
  • blur-radius: 模糊半径，只能是正值
  • spread-radius: 扩展半径，可以是正负值，正值，阴影都延展扩大，负值都缩小
单边阴影，只设置偏移量
box-shadow: 4px 4px #6c6;
单纯设置模糊半径的效果
box-shadow: 0 0 24px 0 #6c3;
单纯设置扩展半径的效果
box-shadow: 0 0 0 24px #6c3;
