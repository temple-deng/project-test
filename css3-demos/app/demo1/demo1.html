语法部分
border-radius: none | <length> {1,4} [/<length>{1,4}]?
如果/存在，则/前面的是元素圆角的水平方向半径，/后面设置的是元素圆角垂直方向的半径
也可以分开写
border-top-left-radius、border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius
注意：如果要重置元素没有圆角，取值为none并无效果，需要将元素的border-radius设置0
还有一点要注意的，只有用缩写写法的border-radius时水平与垂直方向的半径用/分割，如果是分开写各个角的话，就要用空格分割
水平半径于垂直半径不一致的demo
border-radius: 50px/25px;
111
圆形的demo，重点是元素高宽相等，圆角半径等于宽的一半
width:150px; height:150px; border-radius: 75px;
111
半圆形的demo，重点是元素高或宽是另一边的一半，圆角半径也只展示出想要的部分
width:150px; height:75px; border-radius: 75px 75px 0 0;
111
1/4圆形的demo，重点是元素等宽高，圆角半径与宽高长度一致
width:150px; height:150px; border-radius: 150px 0 0 0;
111
椭圆形的demo，重点是元素宽高不等，元素圆角水平与垂直半径也不等
width:150px; height:50px; border-radius: 75px/50px;
